- ADD (registrador):
prefix = 00 = 00000000
opcode = FB = 11111011
descrição: adiciona o valor referente ao registrador dos digitos 2 a 0 ao acumulador (convencionei como o r1)

- ADD (imediato):
prefix = 00 = 00000000
opcode = AB = 10101011
descrição: adiciona o valor imediato dos digitos 7 a 0 ao acumulador (convencionei como o r1)

- SUB (registrador):
opcode = F0 = 11110000
descrição: subtrai o valor referente ao registrador dos digitos 2 a 0 ao acumulador (convencionei como o r1)

- SUB (imediato):
opcode = A0 = 10100000
descrição: subtrai o valor imediato dos digitos 7 a 0 ao acumulador (convencionei como o r1)

- LD (acumulador, endereço):
opcode = B6 = 10110110
descrição: carrega o valor referente ao endereço da ram no registrador acumulador

- LD (endereço, acumulador):
opcode = B7 = 10110111
descrição: carrega o valor referente ao registrador acumulador no endereço de memória especificado

- LDW (registrador, registrador):
opcode = F6 = 11111111
descrição: carrega o valor referente ao registrador dos digitos 2 a 0 no registrador dos digitos 5 a 3


- CLR (registrador):
opcode = 7F = 01111111
descrição: zere o valor referente ao registrador dos digitos 2 a 0

- JPF (memoria):
opcode = AC = 10101100
descrição: pula para o endereço da memória dos digitos 6 a 0

- JRA (memoria):
opcode = 20 = 00100000
descrição: pula para o endereço de memória atual mais o valor carregado

- JRPL (memoria):
opcode = 2A = 00101010
descrição: pula para o endereço da memória atual mais o valor carregado, caso o carry de subtreação seja 0 (caso o primeiro numero seja menor que o outro)
